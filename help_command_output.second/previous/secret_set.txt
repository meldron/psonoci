psonoci-secret-set 0.5.0
Set a psono secret

USAGE:
    psonoci secret set <secret-id> <secret-value-type> <secret-new-value>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

ARGS:
    <secret-id>            The secret's uuid
    <secret-value-type>    Which secret value-type to set ('json' not yet supported) [possible values: json, notes,
                           password, title, url, url_filter, username, gpg_key_email, gpg_key_name, gpg_key_private,
                           gpg_key_public, secret_type, env_vars, ssh_key_public, ssh_key_private, totp_period,
                           totp_algorithm, totp_digits, totp_code, credit_card_number, credit_card_cvc,
                           credit_card_name, credit_card_valid_through, credit_card_pin,
                           elster_certificate_file_content, elster_certificate_password,
                           elster_certificate_retrieval_code]
    <secret-new-value>     The new value to set for type
