psonoci-config 0.5.0
Config commands (create, save, pack,...)

USAGE:
    psonoci --api-key-id <api-key-id> --api-secret-key-hex <api-secret-key-hex> --server-url <server-url> config <SUBCOMMAND>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

SUBCOMMANDS:
    help    Prints this message or the help of the given subcommand(s)
    pack    Pack psonoci config into base58 encoded MessagePack string which can be used by --config-packed
    save    Save psonoci config into a toml file which can be loaded with --config-path
    show    Displays the current config in toml format
