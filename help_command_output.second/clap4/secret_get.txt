Get a psono secret

Usage: psonoci secret get <SECRET_ID> <SECRET_VALUE_TYPE>

Arguments:
  <SECRET_ID>          The secret's uuid
  <SECRET_VALUE_TYPE>  Which secret value-type to return ('json' returns all value-types in a json
                       object) [possible values: json, notes, password, title, url, url-filter,
                       username, gpg-key-email, gpg-key-name, gpg-key-private, gpg-key-public,
                       secret-type, env-vars, ssh-key-public, ssh-key-private, totp-period,
                       totp-algorithm, totp-digits, totp-code, credit-card-number, credit-card-cvc,
                       credit-card-name, credit-card-valid-through, credit-card-pin,
                       elster-certificate-file-content, elster-certificate-password,
                       elster-certificate-retrieval-code]

Options:
  -h, --help  Print help
