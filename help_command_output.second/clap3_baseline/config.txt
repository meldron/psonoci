psonoci-config 
Config commands (create, save, pack,...)

USAGE:
    psonoci config <SUBCOMMAND>

OPTIONS:
    -h, --help    Print help information

SUBCOMMANDS:
    help    Print this message or the help of the given subcommand(s)
    pack    Pack psonoci config into base58 encoded MessagePack string which can be used by
                --config-packed
    save    Save psonoci config into a toml file which can be loaded with --config-path
    show    Displays the current config in toml format
