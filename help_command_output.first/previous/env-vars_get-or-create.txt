psonoci-env-vars-get-or-create 0.5.0
Get or create env var for a specific secret. Will always get the first secret the first secret with the specified name
in the env var list

USAGE:
    psonoci env-vars get-or-create [OPTIONS] <secret-id> <env-var-name>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -a, --danger-password-allowed-chars <allowed_password_chars>
            By default psono uses alphanumeric chars ([a-zA-Z0-9]) for the created passwords.
            This option overwrites the default charset.
            IMPORTANT: Make sure to supply enough chars, otherwise the password will be insecure.
    -n, --password-length <num_chars>
            If a password needs to be created use l chars (unicode graphemes).
            Important: if you are using unicode chars/graphemes with more than one byte per char, the password byte
            length will be bigger than the num of chars [default: 21]

ARGS:
    <secret-id>       The uuid of the secret containing the env var
    <env-var-name>    The name of the env var
