psonoci-totp 0.5.0
TOTP commands

USAGE:
    psonoci --api-key-id <api-key-id> --api-secret-key-hex <api-secret-key-hex> --server-url <server-url> totp <SUBCOMMAND>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

SUBCOMMANDS:
    get-token         Get the current token for a TOTP secret
    get-url           Get the otpauth url for a TOTP secret
    help              Prints this message or the help of the given subcommand(s)
    validate-token    Check if a token is currently valid for a TOTP Secret
