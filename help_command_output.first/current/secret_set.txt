psonoci-secret-set 
Set a psono secret

USAGE:
    psonoci secret set <SECRET_ID> <SECRET_VALUE_TYPE> <SECRET_NEW_VALUE>

ARGS:
    <SECRET_ID>            The secret's uuid
    <SECRET_VALUE_TYPE>    Which secret value-type to set ('json' not yet supported) [possible
                           values: json, notes, password, title, url, url-filter, username,
                           gpg-key-email, gpg-key-name, gpg-key-private, gpg-key-public,
                           secret-type, env-vars, ssh-key-public, ssh-key-private, totp-period,
                           totp-algorithm, totp-digits, totp-code, credit-card-number,
                           credit-card-cvc, credit-card-name, credit-card-valid-through,
                           credit-card-pin, elster-certificate-file-content,
                           elster-certificate-password, elster-certificate-retrieval-code]
    <SECRET_NEW_VALUE>     The new value to set for type

OPTIONS:
    -h, --help    Print help information
