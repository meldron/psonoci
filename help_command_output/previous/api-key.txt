psonoci-api-key 
Psono api-key inspect (/api-key-access/inspect/)

USAGE:
    psonoci api-key <SUBCOMMAND>

OPTIONS:
    -h, --help    Print help information

SUBCOMMANDS:
    help       Print this message or the help of the given subcommand(s)
    info       Prints the meta info of a api-key and lists all its secret ids
    secrets    Prints all secrets of an api-key as JSON
