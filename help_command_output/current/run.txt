Spawns processes with environment vars from the api-keys secrets

Usage: psonoci run [OPTIONS] [COMMAND_VALUES]...

Arguments:
  [COMMAND_VALUES]...  The command you want to run. It's recommended to prefix it with '--' so
                       additional flags won't be interpreted by psonoci

Options:
  -c, --clear-env             Spawn process only with the env vars from the psono datastore
  -f, --filter <secret_uuid>  Only include env vars from secrets explicitly supplied
  -h, --help                  Print help
