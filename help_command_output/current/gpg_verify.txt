Usage: psonoci gpg verify [OPTIONS] --signature <signature path> <SECRET_ID> [INPUT_FILE]

Arguments:
  <SECRET_ID>   The uuid of the secret containing the gpg private key
  [INPUT_FILE]  Input file path. If no path is given, reads from stdin

Options:
  -s, --signature <signature path>  Signature file path
  -q, --quiet                       Do not print verification error
  -v, --verbose                     Print success message
  -h, --help                        Print help
